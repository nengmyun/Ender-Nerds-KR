# 설명서를 읽기 전

:::note[공지]
설명서에 영어 버전과 다른 부분이 있습니다.\
이는 다음에 있을 Unlimited Adventures의 업데이트에 반영될 문서라고 합니다.\
또한 해당 업데이트는 곧 지원될 것이니 굳이 변경할 필요가 없다고 하여 그대로 두겠습니다.

다음 업데이트와 함께 Ender Nerds 공식 Docs에 저희 한글 Docs가 추가될 예정이라고 합니다.\
하지만 여전히 이 설명서는 Ender Nerds의 공식 설명서가 아니므로, 서비스 이용에 참고만 되어야합니다.
:::

해당 설명서는 Ender Nerds의 제품들을 구매하신 한국인분들을 위하여 제작되었습니다.
Ender Nerds의 허락을 받은 후 원본 파일을 제공받아 제작되고 있으며, 상의 후 계속 추가할 예정입니다.
하지만 혼자서 작업하는 것이기에 반영되는것에 다소 시간이 필요합니다.

:::warning[주의]
설명서만 한글화가 진행되고 있으며, Ender Nerds의 모든 부문에서 한국어 서비스를 지원하는 것은 아닙니다.\
또한 아직 한글 번역이 진행중 입니다. 한글이 적용되지 않은 페이지 및 문단이 있을 수 있습니다.
:::

해당 설명서는 한글 번역본 입니다. 영문 버전 설명서는 [여기](https://endernerds.netlify.app/)를 클릭해주세요.


:::info[정보]
한글 번역이 완벽하지 않을 수 있습니다. 만약 오역을 발견하셨다면, 디스코드: dead_kim 으로 친추 후 문의 주세요.
:::
