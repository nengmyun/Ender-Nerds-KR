# Adventure Backpacks 리소스 팩을 ItemsAdder와 병합하는 방법
정말 매우 간단합니다. 글로된 설명을 따르거나 영상을 시청할 수 있습니다!

**Step 1**
> /stop 으로 서버를 멈추세요

**Step 2**
> Adventure Backpacks의 리소스 팩을 ``plugins/ItemsAdder/contents``에 넣고 서버를 킵니다.

**Step 3**
> ``/iazip`` 명령을 실행합니다.

완료! 이제 서버를 시작하고 ItemsAdder와 Adventure Backpacks를 함께 즐겨보세요 ❤️
