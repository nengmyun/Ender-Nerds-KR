# :gift: Crates

Setup comes with 3 pre-configured crates that players can purchase and open.

#### How to get a Crate Key?

You can get a Crate Key by executing `/crazycrates give virtual AdventureCrate 1 [player name]`\
Default crates:
- AdventureCrate
- HeroCrate
- LegendaryCrate
