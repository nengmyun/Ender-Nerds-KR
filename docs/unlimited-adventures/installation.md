# :hatching_chick: 서버팩 설치 가이드

Unlimited Adventures를 구매하셨나요? 현명하시네요! 빨리 이 서버팩을 쓰고 싶으신거 압니다. 그러니, 빠르게 본론으로 가보자고요!
:::warning[경고]
이미 서버팩을 설치하신 후 설정을 끝내셨다면 [업데이트](updating) 페이지로 이동해주세요.
:::

#### Step 1

⬇️ [Paper 1.21](https://api.papermc.io/v2/projects/paper/versions/1.21/builds/130/downloads/paper-1.21-130.jar)를 다운로드 합니다.

#### Step 2
⬇️ 필수 플러그인들을 설치합니다: 
- [탭(TAB)](https://github.com/NEZNAMY/TAB/releases/download/4.1.8/TAB.v4.1.8.jar)
- [시티즌(Citizens)](https://ci.citizensnpcs.co/job/citizens2/lastSuccessfulBuild/artifact/dist/target/Citizens-2.0.35-b3580.jar)
- [모델 엔진(Model Engine)](https://mythiccraft.io/index.php?pages/official-modelengine4-download/&download=free&f=4.0.7)
- [미스틱 몹(Mythic Mobs)](https://www.mythiccraft.io/downloads/mythicmobs/free/MythicMobs-5.7.2.jar)

#### Step 3
설치하신 플러그인을 `plugins` 폴더에 넣습니다.


✅ 어때요, 참 쉽죠? 이제 서버를 즐기시면 되겠습니다!
