# :gift: 랜덤 상자

Unlimited Adventures에는 실제로 사용할 수 있는 사전 제작된 3가지의 랜덤 상자(이하 상자)가 존재합니다.

#### 상자 열쇠를 어떻게 얻나요?

`/crazycrates give virtual AdventureCrate 1 [플레이어 이름]`을 실행하여 상자 키를 얻을 수 있습니다.
사용 가능한 상자: AdventureCrate, HeroCrate, LegendaryCrate

### 상자를 어떻게 수정하나요?

`plugins/CrazyCrates/crates` 폴더에서 상자를 수정할 수 있습니다.
