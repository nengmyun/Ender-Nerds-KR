### :label: 한글 Docs 사이트 교체 안내

영문 Docs 사이트가 대규모 디자인 변화가 생겼기에, 한글 Docs도 영문 Docs의 소스를 기반으로 번역을 다시 진행하려고 합니만, 현재 담당자가 바쁜 관계로 번역 시작엔 조금 시간이 걸릴 수 있습니다.

담당자 변경 혹은 일정 조율등 최대한 빠르게 번역 작업을 진행할 수 있도록 노력하겠습니다.

감사합니다.
