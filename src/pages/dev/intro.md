# 필독사항

해당 문서는 한국어 문서 페이지를 관리하는 사람들을 위해 제작된 문서 페이지 입니다.
공지 작성 형식 등을 찾아볼 수 있습니다.

:::warning[주의]
해당 문서 페이지는 Ender Nerds 한국어 문서 페이지의 개발 및 운영자들을 위한 페이지 입니다.
일반 유저분들의 진입을 고려하지 않았기에 문서를 확인하는 것에 있어 불편함이 있을 수 있습니다.
일반 문서 페이지는 [이곳](/docs/intro/)를 참고해주세요.
:::
