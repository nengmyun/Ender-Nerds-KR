# # 언리미티드 어드벤처 / 2.7.2 - 분석 업데이트
## 분석
매일 새로 가입하는 플레이어에 대한 데이터 덕분에 서버의 성과를 측정할 수 있습니다! 이것은 업계 표준이며, 귀하의 서버를 위해 고안한 귀하의 서버 진행 상황을 추적하기 위한 실행 가능한 지표입니다!

## 변경 사항​
- 이제 뱃자에서 현재 진행 과정이 표시됩니다.
- 텔레포트 명령이의 기본 설정이 활성화로 변경됩니다.(현재 사용자에게는 영향을 미치지 않음)
- 이제 첫 접속 시 자동으로 나가지지 않습니다. 이제 리소스 팩이 올바르게 적용되어 플레이어의 경험을 원활하게 유지합니다!
