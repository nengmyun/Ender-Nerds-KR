# # 언리미티드 어드벤처 / 2.8 - 여정 업데이트

## 🌳 바이옴 및 구조물
* 주변 환경을 완벽하게 지원하는 수 많은 새로운 바이옴
* 80개의 새로운 바이옴 추가
* 40개의 새로운 구조물 추가. (새로운 마을, 약탈자 전초기지, 네더 요새 등 포함)
* 14개의 동굴 바이옴 추가

## ⭐ 1.21 지원
* 이제 Unlimited Adventures는 1.21을 배이스로 구동됩니다. (1.21에 추가된 아이템 이용 가능)

## 🎵 소리
* 동굴 탐험 시 몰입감을 올려줄 자연스러운 동굴 환경음을 추가
* 고유한 도구 착용 사운드 추가
* 핫바 슬롯(1~9번 칸) 이동 사운드 추가
* 인벤토리에서 아이템을 옮기는 소리 추가 (이동하는 아이템의 타입에 따라 다른 소리 재생)
* 채팅에서 메시지가 전송될 때 소리 추가
* **The Forge**에서 플레이어가 아이템을 제작할 때 출력되는 커스텀 사운드 추가
* 플레이어가 상점 메뉴에서 아이템을 구매할 때 출력되는 커스텀 사운드 추가
* 플레이어가 프리미엄 스토어에서 무언가를 구매할 때 출력되는 커스텀 사운드 추가
* 플레이어가 제작대에서 무언가를 제작할 때 출력되는 커스텀 사운드 추가
* 플레이어가 상호작용하는 모든 블럭(용광로, 제작대 등)을 열 때 출력되는 커스텀 사운드 추가

## 📊 Ul 개선
* 해가 뜨면 서버에서 플레이한 일(day)수가 귀여운 타자기 효과로 표시됩니다.
* 바이옴에 들어가면 화면에 바이옴이 표시됩니다.
* /getitems 명령은 이제 페이지별로 구분되어 모든 사용자 정의 항목이 자동으로 표시됩니다.
* /day 및 /night 명령에 이제 멋진 '타임랩스' 효과가 추가되었습니다.

## 😄 플레이 퀄리티
* 완전히 자란 작물을 마우스 오른쪽 버튼으로 클릭하면 수확이 가능하며, 씨앗이 충분하다면 즉시 다시 심습니다.

## 🔧 밸런스 패치
* 열기구에서 네더라이트 조각이 더이상 등장하지 않습니다.
* 이제 드래곤스케일 아이템을 제작하려면 네더라이트 아이템이 기본으로 필요합니다.

## 🔧 기타 변동사항
* 일부 아이템 착용 시 빤짝이 파티클이 나타납니다.
* 채팅 플러그인을 ChatManager에서 TChat으로 변경했습니다.
* Forge 시스템을 더욱 강력하게 만들고 오류 발생 가능성을 줄였습니다.
* 앉는 자세를 개선하였습니다.
* 기타 변경 사항 및 버그 수정

:::info[업데이트 버전 호환]
해당 업데이트는 2.7.2 버전에서 업데이트 가능합니다.
