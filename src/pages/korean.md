# **:kr: Unlimited Adventures의 한글화 관련 공지사항입니다!**
새로운 번역 지원자는 없었지만 작업 시작이 확정되었으며, 정확한 작업 일정을 알려드리려고 합니다!\
Unlimited Adventures의 한글화 작업은 다음 대규모 업데이트는 2.7 버전부터 작업이 시작됩니다.
:::tip[2.7버전 부터 시작하는 이유]
2.6 버전은 이미 나온지 꽤 된 버전입니다.
지금 작업을 시작할 시 2.7 버전이 나와있을 가능성이 매우 크므로 2.7 버전이 나오자마자 작업하여 계속 업데이트를 지원하는 방식으로 될 예정입니다.
:::
다음으로는 궁금하실 것 같은 부분에 대한 답변입니다.

:::note[1. 한글 패치는 별도로 구매해야하나요?]
아닙니다! Unlimited Adventures를 구매하셨다면, 무료로 제공 받으실 수 있습니다.
:::
:::note[2. 한글 패치는 어떻게 적용하나요?]
버전을 업데이트하는 것 처럼 한글 버전 파일을 적용하는 방식입니다!
:::
:::note[3. 한글 패치는 꾸준히 업데이트 되나요?]
아직 후속 패치에 관한 내용은 논의된 것이 없습니다.
:::
:::note[4. 한글 패치는 어디서 다운해야할까요?]
Unlimited Advenutres를 다운받으신 파일에 동봉될 예정입니다.
:::
:::note[5. 어디까지 한글 패치가 진행되나요?]
GUI, 기타 리소스팩을 제외한 인게임 속 영어들은 최대한 한글패치 할 예정입니다.
:::
:::note[6. 언제쯤 출시되나요?]
아직 한참 멀었습니다만, 최대한 빠르게 진행할 예정입니다.
:::
